# Sky-Center-JS
Skype + Call Center

#Requirements:
* Nodejs
* NPM

#Install:
Clone repo:

>git clone https://github.com/Pooler22/Sky-Center-JS.git

Create file config.json, with skype account name and password:

>{
  "user": "username",
  "password": "password"
}

Install dependencies:

>npm install

Run:

>node app.js
